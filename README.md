# Meet Gojiya Portfolio

Modern React portfolio site built with Vite and Framer Motion.

## Run Locally

```bash
npm install
npm start
```

The dev server starts with Vite and prints a local URL in the terminal.

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Rollback

The redesign work is on branch `codex/modern-portfolio-redesign`.  
To return to the previous version, switch back to `main`.
